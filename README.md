# 个人简历网页

极简主义风格的个人简历展示网页，采用纯黑白灰三色设计。

## 项目结构

```
个人简历网站/
├── index.html          # 主页面
├── SPEC.md             # 技术规范文档
├── README.md           # 项目说明文档
├── css/
│   ├── variables.css   # CSS 变量定义
│   ├── reset.css       # CSS 重置样式
│   ├── base.css        # 基础样式
│   ├── layout.css      # 布局样式
│   └── components.css  # 组件样式
└── js/
    ├── navigation.js   # 导航功能模块
    ├── scroll.js       # 滚动控制模块
    └── app.js          # 主入口文件
```

## 技术规范

### 四层分层架构
- **全屏舞台层**：占满 100% 视口
- **背景层**：承载背景装饰
- **内容层**：页面具体内容
- **全局固定 UI 层**：悬浮导航、翻页箭头、指示器

### 颜色系统
| 用途 | 色值 |
|------|------|
| 全局背景 | #191919 |
| 辅助元素 | #4c4c4c |
| 核心内容 | #ffffff |

### 字体系统
- 字体：微软雅黑
- 字重：Light / Regular / Bold

## 页面模块

1. **首页** - 个人主视觉区
2. **关于我** - 个人介绍卡片
3. **专业技能** - 技能熟练度展示
4. **工作经历** - 时间线展示
5. **作品展示** - 项目卡片网格
6. **联系方式** - 联系信息 + 表单

## 使用说明

1. 直接用浏览器打开 `index.html` 即可预览
2. 编辑 `index.html` 中的中文注释标注内容进行个性化定制
3. 各页面模块可独立修改或删除，不影响其他模块

## 交互功能

- 全屏滚动切换页面
- 悬浮导航点击跳转
- 页面指示器导航
- 上下翻页箭头
- 键盘方向键导航
- 触摸滑动支持

## 自定义内容

所有需要替换的内容在 HTML 中都有中文注释标注，例如：
- `<!-- 【需要替换】姓名 -->`
- `<!-- 【需要替换】职位名称 -->`

找到对应位置修改即可。
