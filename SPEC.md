# 个人简历网页 - 技术规范文档

## 1. 概念与愿景

一个极简主义风格的个人简历展示网页，以深黑色为主基调，通过纯黑白灰三色营造高端、专业、沉稳的视觉感受。页面采用全屏滚动式单页应用设计，每个页面模块独立完整，通过悬浮导航实现流畅的页面切换体验。

## 2. 设计语言

### 美学方向
- 极简主义黑白设计，强调留白与排版
- 参考高端品牌官网的简洁风格

### 色彩系统
| 用途 | 色值 | 说明 |
|------|------|------|
| 全局背景 | #191919 | 深黑色，用于所有页面、卡片、容器背景 |
| 辅助元素 | #4c4c4c | 中灰色，用于辅助文字、分割线、边框、未激活状态 |
| 核心内容 | #ffffff | 纯白色，用于标题、核心内容、激活状态、按钮文字 |

### 字体系统
- 字体族：Microsoft YaHei（微软雅黑）
- 字重规则：
  - Light (300)：大标题装饰性文字、最次要说明文字
  - Regular (400)：正文、二级标题、按钮文字、联系方式
  - Bold (700)：页面主标题、核心关键词、需要强调的内容

### 空间系统
- 设计基准宽度：1920px
- 内容区最大宽度：1440px
- 页面顶部内边距：60px
- 模块间距：80px
- 元素间距：24px

### 动效哲学
- 页面切换：平滑过渡，0.6s ease-out
- 元素进入：淡入上移，opacity 0→1 + translateY(30px→0)，0.8s ease-out
- 悬浮效果：轻微放大 scale(1.02)，0.3s ease
- 导航指示器：平滑滑动，0.4s ease

## 3. 布局与结构

### 四层分层架构
```
┌─────────────────────────────────────┐
│         全局固定 UI 层 (z-index: 100)│  ← 悬浮导航、翻页箭头、指示器
├─────────────────────────────────────┤
│           内容层 (z-index: 10)       │  ← 各页面具体内容
├─────────────────────────────────────┤
│           背景层 (z-index: 1)        │  ← 背景装饰元素
├─────────────────────────────────────┤
│         全屏舞台层 (z-index: 0)      │  ← 100vw x 100vh 容器
└─────────────────────────────────────┘
```

### 页面模块（内容层）
1. **首页/关于我** - 个人简介、技能概览
2. **经历** - 工作经历、教育背景
3. **作品** - 项目/作品展示
4. **联系方式** - 联系方式信息

### 响应式策略
- 优先使用 Flexbox 和 Grid 布局
- 减少绝对定位使用
- 禁止横向滚动条

## 4. 功能与交互

### 核心功能
- 全屏滚动切换页面
- 悬浮导航菜单（可点击跳转）
- 页面指示器（显示当前位置）
- 翻页箭头（上下翻页）
- 键盘导航支持（方向键）

### 交互细节
- 导航项悬浮：文字颜色变亮 + 下划线动画
- 页面切换：内容淡入动画
- 指示器点击：直接跳转到对应页面
- 滚动检测：自动更新当前页面状态

## 6. 技术方案

### 技术栈
- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖
- 无外部字体文件

### 文件结构
```
个人简历网站/
├── index.html          # 主页面
├── css/
│   ├── reset.css       # CSS 重置
│   ├── variables.css   # CSS 变量定义
│   ├── base.css        # 基础样式
│   ├── layout.css      # 布局样式
│   └── components.css  # 组件样式
├── js/
│   ├── app.js          # 主入口文件
│   ├── navigation.js   # 导航功能
│   └── scroll.js       # 滚动控制
└── SPEC.md             # 本规范文档
```

### 模块化开发原则
- CSS 使用后代选择器隔离样式
- JavaScript 使用独立作用域，不污染全局
- 每个模块前后加注释分隔线
- 可独立删除任意模块而不影响其他模块
